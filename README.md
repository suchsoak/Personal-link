## 🚀 Funcionalidades

O projeto permite criar uma landing page minimalista com:

- 📱 **Design Responsivo**: Adaptado para dispositivos móveis e desktop.
- 🔗 **Agregador de Links**: Botões personalizados para todas as suas redes sociais.
- 🎨 **Estilização Simples**: Fácil customização de cores e fontes.
- ⚡ **Performance**: Carregamento rápido por ser uma página estática.

## ⚙️ Como Personalizar

1. Clone o repositório:
   
   ```bash
   git clone https://github.com/suchsoak/Personal-link.git
   ```

3. Edite os links e textos no arquivo principal (ex: `index.html`.)
4. Altere o arquivo CSS para modificar as cores e o layout conforme seu estilo.
